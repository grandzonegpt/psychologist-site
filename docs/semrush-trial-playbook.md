## День 1-2: Backlink Gap (самое ценное)

Шестаков: «**Links rule. Если нет ссылок, ты в жопе.**» Backlink Gap показывает где у конкурентов беклинки, которых нет у нас. Это outreach-карта на месяцы.

### Конкуренты для RU-аудитории

В `Backlink Analytics → Backlink Gap` ввести через запятую:
- `levashou.pl` (твой)
- `alter.app` (российский маркетплейс терапевтов, многие эмигранты)
- `yasno.live` (украинский, активно растёт)
- `b17.ru` (старый российский справочник)
- `meta.app` (если живой)

Получишь список **доменов которые ссылаются на конкурентов, но не на тебя**. Сортируй по Authority Score, экспортируй топ-50 в CSV.

Тип ссылающихся, которые ищем:
- RU-блоги про эмиграцию (telegram-каналы, медиум-блоги)
- RU-медиа про ментальное здоровье (вмире.media, Mel.fm, batenka.ru, mednet.tilda и т.д.)
- Польские RU-комьюнити (форумы, FB-группы)
- Listicles «лучшие психологи онлайн»

### Конкуренты для PL-аудитории

Отдельный запуск `Backlink Gap`:
- `levashou.pl`
- `znanylekarz.pl` (хотя это агрегатор, всё равно полезно)
- `helloterapia.pl`
- `psychoterapia24.pl`
- `mentalio.pl`

## День 2-3: Keyword Gap (контент-план)

`Keyword Gap` → `levashou.pl` + 3 конкурента → покажет ключи где конкуренты в топ-10, а ты нет.

### Фильтры

- **KD (Keyword Difficulty) ≤ 30** — что реально пробить молодому домену
- **Volume ≥ 50** — есть смысл писать
- **Intent: Informational** — для блога

Экспортируй CSV. Это контент-план на 3-6 месяцев. Каждая строка = потенциальный блог-пост.

### Для RU

Конкуренты: `alter.app/blog`, `yasno.live/blog`, `b17.ru/articles`, `psy.systems`

### Для PL

Конкуренты: `helloterapia.pl/blog`, `mentalio.pl/blog`, `mp.pl/pacjent/psychiatria`

## День 3-4: Site Audit (второе мнение)

`Site Audit` → `levashou.pl` → Run. Через 10 минут получишь:
- Health Score (наш должен быть 90+)
- Список ошибок: сломанные ссылки, дубли title, missing alt, медленные страницы
- Сравни с тем что уже сделано — найди что пропустили

Особо смотри:
- **Crawled - currently not indexed** — какие страницы Google знает но не показывает
- **Page with redirect** — http→https нормально, остальные смотри руками
- **Duplicate title/desc** — наша слабая точка возможно

## После триала: free tools

Что заменит Semrush:

### Position Tracking
- **Serpstat** — есть free план с 10 ключами/день
- **AccuRanker** — есть free trial
- **Google Search Console** (бесплатно) — но без daily tracking, только месяц-к-месяцу

### Backlinks
- **Ahrefs Webmaster Tools** (бесплатно для своего сайта) — покажет твои беклинки
- **Google Search Console → Links** — то же
- **Для анализа конкурентов**: ahrefs.com/free-backlink-checker, neilpatel.com/ubersuggest

### Keyword research
- **Google Keyword Planner** (нужен Google Ads аккаунт, не обязательно платить)
- **AnswerThePublic** (бесплатно с лимитом)
- **Google Search → People Also Ask + Related Searches**

### Site Audit
- **Google Search Console** + **PageSpeed Insights** + **Rich Results Test**
- **Screaming Frog SEO Spider** (free до 500 страниц)
