# Google Ads: пилот на индивидуальную терапию

Подготовлено 2026-06-11. Фокус: индивидуальные сессии (пары исключены по решению Алексея).
Бюджет пилота: 600-700 PLN/мес (~20-23 PLN/день суммарно). Срок оценки: 4 недели.
Цель оптимизации: конверсия `intro_booking` (запись на бесплатные 15 минут), не клики.

## Промо-купон Google Ads (новый аккаунт, Польша)

При регистрации Google даёт купон «потрать X — получишь X бонусом» (пороги 1200 / 3500 / 7000 PLN).
Это НЕ деньги на старт: бонус приходит только после того, как кампании реально открутят
порог за 60 дней; сам бонус тоже надо потратить за 60 дней, иначе сгорит. НДС 23% на тебе.

- **Активировать купон по ссылке из письма ДО/во время регистрации** (цепляется к первому аккаунту,
  потом не прицепить).
- **Брать порог 1200**, не 3500/7000: при бюджете ~690 PLN/мес порог 1200 добирается за 2 месяца
  естественно, без раздувания. Высокие пороги заставят тратить втрое больше плана.
- **Не увеличивать бюджет ради бонуса.** Правило выключения ниже (intro дороже 120 PLN → стоп)
  сильнее купона. Бонус не меняет решение «работает / не работает».
- После бонуса Google списывает с карты дальше: держать лимиты, можно ставить на паузу в любой момент.

## Перед запуском (один раз, в кабинетах)

1. Связать Google Ads ↔ GA4: Ads → Инструменты → Связанные аккаунты → Google Analytics.
2. Конверсии. Виджет уже шлёт два события: `intro_booking` (бесплатное знакомство, главная
   цель пилота) и `purchase` (платная сессия). Есть два способа их считать, лучше включить оба:

   **(а) Прямой тег Ads (точнее и мгновенно, нужен для smart bidding).** Создай в Ads две
   конверсии: Ads → Цели → Конверсии → Новая → «Website», вручную:
   - «Intro 15 min» (категория «Submit lead form», ценность не задавать, count: One),
   - «Paid session» (категория «Purchase», ценность 180 PLN).
   Скопируй из каждой её Conversion Label и Tag ID (AW-XXXXXXXXXX) и впиши в сайт. Один файл,
   в начале `js/main.js`, блок `window.MARKETING_IDS`:
   ```
   googleAdsTag:             'AW-XXXXXXXXXX',
   googleAdsConversionLabel: 'лейбл_платной_сессии',
   googleAdsIntroLabel:      'лейбл_intro_15min',
   ```
   После правки сказать мне: пересоберу main.min.js и подниму версию (иначе CDN отдаст старый).
   Технически событие на стороне сайта уже подключено к обоим лейблам, ждёт только ID.

   **(б) Импорт из GA4 (запасной, медленнее на несколько часов).** Ads → Цели → Конверсии →
   Импорт → GA4 → отметить `intro_booking` и `purchase`. Полезен как дубль-страховка.

   Основной для оптимизации в кампаниях ставить «Intro 15 min».
3. В настройках кампаний выключить: КМС (display), поисковых партнёров.
4. Автотегирование (gclid) оставить включённым. Ручные UTM в ссылках ниже уже расставлены,
   они дадут источник в нашей собственной атрибуции booking-api.

## Скрипт еженедельного отчёта (Ads → Инструменты → Скрипты)

```javascript
function main() {
  var stats = AdsApp.currentAccount().getStatsFor('LAST_7_DAYS');
  var rows = [];
  var it = AdsApp.campaigns().forDateRange('LAST_7_DAYS').get();
  while (it.hasNext()) {
    var c = it.next(); var s = c.getStatsFor('LAST_7_DAYS');
    rows.push(c.getName() + ': клики ' + s.getClicks() + ', расход ' +
      s.getCost().toFixed(0) + ' PLN, конверсии ' + s.getConversions());
  }
  MailApp.sendEmail('goalcoachup@gmail.com',
    'Ads за неделю: ' + stats.getConversions() + ' записей, ' + stats.getCost().toFixed(0) + ' PLN',
    rows.join('\n'));
}
```
Расписание: еженедельно, понедельник 09:00.

## Критерий месяца

Цена одной записи на intro ≤ 60-70 PLN: масштабируем победителя.
Дороже 120 PLN при ≥200 кликах: кампанию выключаем, выводы в семантику SEO.
