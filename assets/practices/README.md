# assets/practices

Сюда кладём визуальные шаблоны/знаки для раздела «Практики» (фото, PNG, SVG).
Стиль: тёплый бежевый фон, оливковые/сепийные тонкие линии, без людей и медицинских иконок.

## Куда что попадает (имя файла → где используется)

Индексная страница `/practices.html`:
- `hero-sign.*`      — деликатный знак в hero (дыхательная линия / круг / сетка)

Декоративные микро-знаки в углу карточек и в hero страниц практик
(если положишь файл с таким именем — заменю инлайновый SVG на него):
- `breath-line.*`     — дыхательные практики (4-7-8, box breathing)
- `grounding-radar.*` — заземление 5-4-3-2-1
- `calm-grid.*`       — заземление / сетка (альтернатива радару)
- `thought-dot.*`     — дневник тревожных мыслей
- `abc-steps.*`       — ABC-разбор
- `step-mark.*`       — общий маркер шага
- `body-wave.*`       — мышечная релаксация (PMR)
- `scale-line.*`      — общий декоративный мотив (линия-шкала)
- `soft-orbit.*`      — общий декоративный мотив (круг с точкой)

## Привязка к страницам
| Практика | Страница | Знак |

| 4-7-8 дыхание | breathing-4-7-8.html | breath-line |
| 5-4-3-2-1 заземление | grounding-5-4-3-2-1.html | grounding-radar / calm-grid |

Форматы: SVG предпочтительно (чёткость, малый вес), либо PNG с прозрачным фоном.
Цвет линий желательно `#8a6a3a` (сепия-акцент) или нейтральный, чтобы наследовать `currentColor`.
